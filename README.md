# CreatingArtOnline.com

Free online digital drawing studio with enterprise-grade security.

## Security Features

- ✅ HTTPS enforced (HSTS)
- ✅ Content Security Policy (CSP)
- ✅ X-Frame-Options (Clickjacking protection)
- ✅ X-Content-Type-Options (MIME sniffing protection)
- ✅ Referrer Policy
- ✅ Permissions Policy
- ✅ XSS Protection
