# ArtFlow Studio

A professional digital art application combining the best features of Art Set 4 and FlipaClip Studio. Built with pure HTML5 Canvas and vanilla JavaScript.

![ArtFlow Studio](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Realistic Brushes**: Oil, watercolor, pencil, marker, and custom skin texture brushes
- **Gemstone Effects**: Ruby, sapphire, emerald, diamond, amethyst, and topaz rendering
- **Layer System**: Full layer management with opacity and visibility controls
- **Selection Tools**: Rectangular and lasso selection with move capability
- **Light Editing**: Brightness, contrast, saturation, and blur adjustments
- **Mobile Optimized**: Touch-friendly interface for phones and tablets

## Usage

Simply open `index.html` in any modern browser. No build process required!

## Keyboard Shortcuts

- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Y` - Redo

## Tech Stack

- HTML5 Canvas API
- Vanilla JavaScript (ES6+)
- CSS3 with Glassmorphism UI
- No external dependencies
